### V1.0
* 组件是[moment.js v2.5.1](http://momentjs.com/)的KISSY Gallery封装。简单修改以符合KISSY规范。
* 国际化功能，只保留了`zh-cn`(简体中文)、`zh-tw`(繁体中文)、`en`(英文)三种语言。
* 添加了基本的中文demo和文档，更多方法可以阅读[moment.js官方文档](http://momentjs.com/docs/)
